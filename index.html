# Without seed — different results each run
r1 = np.random.rand(3)
print("Without seed:", r1)

# With seed — same results every run
np.random.seed(42)
r2 = np.random.rand(3)
print("With seed 42:", r2)

r3 = np.random.rand(3)

np.random.rand(n)Generates n random floats in [0.0, 1.0)

SyntaxExplicit for/while loopsDirect array operations (+, *, etc.)
MemoryLess efficientContiguous memory, cache-friendly
ReadabilityVerboseConcise and clean

Output

📘 Explanation

abs(x) applied to a NumPy array computes element-wise absolute values: |-2|=2, |-1|=1, |0|=0, |1|=1, |2|=2.

Output
[11 12 13] [[11 21 31] [12 22 32] [13 23 33]] (3, 4)

📘 Broadcasting Rules

Broadcasting allows NumPy to operate on arrays of different shapes without copying data:

RuleDescription
Rule 1If arrays differ in number of dimensions, prepend 1s to the smaller shape
Rule 2Arrays with size 1 along a dimension are stretched to match the other
Rule 3Dimensions must be equal or one of them must be 1; otherwise error

📘 Universal Functions (ufuncs) — Explained

NumPy ufuncs (Universal Functions) are vectorized wrappers around C-level operations that execute element-wise on arrays without Python loops — making them extremely fast.

ufuncCategoryPurpose
